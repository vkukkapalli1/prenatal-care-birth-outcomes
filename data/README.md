# Data

The raw natality CSV files are not included in this repository.
This project uses U.S. natality data from the National Center for Health Statistics (NCHS), covering 2017–2023. 
The original data included more than 25 million observations. 
For this analysis, 70,000 observations were sampled, with 10,000 observations from each year.
Raw data files are excluded because of file size and data management considerations. 
The repository includes the R Markdown analysis file, rendered HTML output, final written report, and presentation slides so the project workflow can be reviewed.
